# 7-Day Average

## Description
- This program uses the New York Times repository of live, cumulative COVID cases to create a 7-day average and compares this week's average to the previous week.

## How to Run
- Copy program into project path with python interpreter.

## Usage 
- After executing file, input the full name of a U.S. state or territory (including U.S. territories), with the first letter capitalized. You may input multiple states. The output will display the most recent 7-day average of COVID cases, and the rate of increase or decrease compared to the prior week
